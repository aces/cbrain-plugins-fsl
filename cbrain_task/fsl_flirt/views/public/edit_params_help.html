FLIRT options

Not all options listed are provided on CBRAIN
To request additional options, leave feedback.


This tool will performs FLIRT of FSL.

Options:

    Input files

  • Input (-in): name of input file.
  • Reference (-ref): name of reference file.

    • Output file

  • Output (-out): name of output file.

    • General options

  • Model/DOF (-dof): restrict the transformation type.
    In 2D to 2D mode only 3 DOF transformations are allowed.
    For 3D to 3D mode the DOF can be set to 12, 9, 7 or 6.
  • Search Angles (-searchx, -searchy, -searchz ): select the angular range over which the initial optimisation search stage is performed.
    Must be between -180 and 180 for X-axis, Y-axis, Z-axis.
  • Cost Function (-cost): select the desired cost function.
  • Bins (-bin): number of histogram bins
  • Interpolation (-interp): final interpolation method.
